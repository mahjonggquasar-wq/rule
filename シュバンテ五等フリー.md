
# シュバンテ五等

- 東風戦、アリアリルール、ツモ損無し、ツモり切り
- 修羅場（局が進むと本場が必ず増える）
- オーラス親の続行は任意
- ドラは常時2枚めくり、南家がドラ表示牌をめくる
- 赤青黒黒赤は祝儀青は裏ドラ
- 完全順位制 0枚 -40枚 or 15枚 -25枚
- 8万点4枚、9万点6枚、10万点10枚
- 北は抜きドラ（手牌で使えるのは国士無双、字一色、四喜和のみ）
- 1本場2000点、ツモ1000点オール
- ノーテン罰符は場に2000点
- 金星、大金星あり
- 白ポッチはリーチ後永久、アガリでチューリップ
- リーチ後の河遊びアリ (トマト, オセロ, キツツキ, シンブンシ, ビンゴ, テトリス)

### 【副露リーチ】

- 副露リーチをすることができる。副露リーチは0翻。
- 1翻が確定していない場合オープンリーチ（全開け）が必要。
- 副露リーチのフリテンは必ずオープン、リーチ後の見逃し無し。

### 【季節牌特殊効果】

- **春**：抜いたその場で抜いている季節牌の数だけ祝儀がもらえる。
- **夏**：アガりがワンランクアップ。役満は5倍満に昇格
- **秋**：5がワンランクアップ。(黒→ドラ1 赤青→ドラドラ)
- **冬**：上下段アリス

※ 季節牌が表示牌に現れた時はアガった人が発動する。裏ドラも同様。アリスに現れたときは抜いている花が対象になる。

### 【特殊役】

- 槓は1翻UP
- 流し役満、リーのみ(季節牌無し)、オープンのみ(出和了限定)、四連刻、萬子の混一色、清一色七対子は純正役満
- チャンタ系の翻数は2倍
- 三連刻(2翻)、三風(2翻)、4枚七対子(4翻)、4枚使い2組七対子(6翻)

### 【祝儀1枚】
一発 / 裏ドラ / 北 / アリス

### 【祝儀3枚】
赤牌 / 金星 / トリプル

### 【その他】
大金星（5枚） / 数え役満（5枚） / 数え役満夏（8枚)
純正役満(10枚) / トバし（10枚） / 純正役満夏（15枚） / 4花4北（50枚）

### 【出目金】
純正役満、4花（ドラ表示OK）、4北、即白ポッチ

### 【シュバリーチ】
1ゲームに1度シュバ棒を使い、獲得祝儀UPのリーチを打つことが出来る
- **シュバ**：シュバ棒＋1000点で祝儀2倍
- **シュバゾーマ**：シュバ棒＋10000点で祝儀3倍
- **シュバンテ**：シュバ棒＋60000点以上の全点棒で祝儀10倍
- 一度使ったシュバ棒は消滅する。
